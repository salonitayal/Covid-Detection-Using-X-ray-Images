This project deals with detection of Covid-19 using X-Ray images. We have used tensorflow as a backend for developing the Machine learning model, keras for preprocessing and accessing the X-ray image dataset, and flask to build the app. We have embed the ML model into our flask app and created the web version of the Machine learning model.
Note: This project is developed for study purposes only.
